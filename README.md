## Start

- copy "run_start.sh" to the project root
- sh run_start.sh
- CHANGE APP_URL in .env


## Dashboard

- /auth/login
- Login: ADMIN_EMAIL in .env
- Password: ADMIN_PASSWORD in .env
- CHANGE auth data in dashboard after log in !!!
- CHANGE ADMIN_TOKEN in .env !!!
